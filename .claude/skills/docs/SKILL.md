# Documentation Generator

Generate or update README and project documentation.

## Usage
```
/docs readme-v1     # Write README with preliminary (WSL 10%) results
/docs readme-v2     # Update README with final (Colab full) results
/docs architecture  # Write architecture explanation section
```

## README Structure

1. Project title, badges, overview
2. CNN Fundamentals (15-year-old level explanations)
3. Architecture explanations (6 models with diagrams)
4. Architecture comparison table
5. Transfer learning explanation (3 stages)
6. Results & Analysis (preliminary → final)
7. Celebrity experiment gallery
8. Installation & usage (WSL + Colab)
9. Project structure
10. Insights & conclusions

## README v1 (Preliminary)
- Written after WSL 10% testing
- All results labeled "Preliminary (10% subset)"
- Full educational content (CNN concepts, architectures)
- Placeholder for final Colab results

## README v2 (Final)
- User pastes Colab Results Export JSON
- Replace preliminary numbers with final results
- Add celebrity/animal experiment sections
- Add final insights and conclusions
